# INFO
# WebServer + RestServer + Mongo

```
Este proyecto utiliza las nociones básicas de un rest server con MVC. 
Para correrlo localmente generar un archivo en la raíz .env que contenga PORT=XXXX donde XXXX es algún puerto disponible que poseas.
La app se puede ejecutar con npm install para construir los modulos de node y luego con el comando npm start
```
